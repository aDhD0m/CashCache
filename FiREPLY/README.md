# MarketPIREP

Social sentiment analysis platform. Separate spec. Not part of TALON.

TALON integration point: `IntelligencePort<SentimentState>`
When absent: Modules ignore sentiment entirely.

## Subsystems (defined in TALON S16.3)

- Sky Spotters -- panelists
- The Runway -- discussion queue
- Atop the Horns / Beneath the Claws -- bull/bear consensus
- Holding Pattern -- low-confidence quarantine
- Boneyard -- discarded signals
- Grounded -- demoted panelists
